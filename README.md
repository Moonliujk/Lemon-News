## Features
- 具有下拉刷新功能
- 可以通过“柠檬助手”进行新闻类型的选择（可以选择：国内、国外、娱乐、体育、军事、财经及其他类别新闻）
## Characters
- 以柠檬黄为主题色调构建新闻小程序
- 新闻列表UI设计，方便概览新闻基本要素
- 独特的“柠檬助手”在交互时十分便捷
## Source
- 本项目来自于 Udacity 微信小程序课程
